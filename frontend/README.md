# Fitness-Agent Frontend

移动端优先的 Vue 3 H5 前端原型，用于论文项目《融合知识增强与动态规划的健身智能体研究——基于 Qwen2.5 大模型的实现》。

## 运行命令

```bash
cd frontend
npm install
npm run dev
```

默认访问地址：

```text
http://localhost:5173
```
